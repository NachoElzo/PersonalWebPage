How to Install chat GPT in visual code

  1. Go to extension options in visual code
  2. Install "code GPT" extension (official openAI)
  3. Go to options/settings
  4. Search for "code gpt" for configurations
  5. Get the API in the API key link
  6. Login openAI and create the new key
  7. Add a key name and you will generate the API Key, copy the key
  8. open a new command palette view/command palette
  9. Search for CodeGPT: set API KEY
  10. Paste the API key and reload visual code
  11. Search for "ask GPT" and ask for a code inside the page that you want to create or modify the code

Node version manager nvm

  • NVM documentation here
  • for install --> terminal: curl -o- https://raw.githubusercontent.com/nvm-sh/nvm/v0.39.7/install.sh | bash
  • In order to avoid the error zsh: command not found: nvm type in the terminal: source ~/.nvm/nvm.sh
  • open and close your terminal
  • NVM commands examples:
  • - Check versions availables for installation --> nvm ls-remote
  • - Install latest version of node --> nvm install node
  • - Install node version --> nvm install "version"
  • - Use node version --> nvm use "version"
  • - List of node version installed --> nvm list
  • - Set default version --> nvm use default "version"
  • - Node version active -->node -v
  • - nvm uninstall "version"

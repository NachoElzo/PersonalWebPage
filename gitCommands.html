Git Commands

  • add a new remote repository: git remote add origin remoteURL
  • add a new remote repository: git remote set-url origin remoteURL
  • deletes a file from the stage: git rm --cached filename (push changes to delete from the repo)
  • Removes ALL files from the stage: git rm -rf --cached . (push changes to delete ALL from the repo)
  • if your .gitignorefile is not working use: git rm -r --cached .
  • ignore files: fileName -- folders: foldername/ -- path: folderName/folderName
  • force changes even if you didnt' pull correctly git push --force origin main
